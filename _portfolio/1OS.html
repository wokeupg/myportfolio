--- title: "Linux-Based OS" excerpt: "Built a basic operating system from scratch in a team of 4.
" collection: portfolio --- In the Fall of 2024, I collaborated in a team of 4 to build a Linux-based operating system from scratch. The OS implemented memory paging and virtual memory, a read-only file system, and multiple device drivers, including keyboard, mouse, and real-time clock. It supported executing user-level programs via system calls and featured multi-task scheduling for concurrent process execution. I primarily contributed to initializing the Interrupt Descriptor Table (IDT), setting up paging, implementing the real-time clock, writing system call operations, and developing the file system driver.

OS Features

  • Memory paging / virtualization – initialized paging
  • Terminal I/O and RTC – implemented terminal input/output and real-time clock
  • In-memory filesystem – read-only file system with file lookup and directories
  • User/Kernel Privilege Mode – enforced via system calls
  • User programs – loaded and executed user-level programs
  • Multitasking – task management with Process Control Blocks (PCBs)
  • Multiple active shells – multi-terminal functionality
  • System call interface – enabled user programs to request kernel services
  • Scheduling – multi-task scheduling across terminals
